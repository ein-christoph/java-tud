Self Study

Learn and practise advanced Java programming with INLOOP

If you want to study more advanced Java programming yourself or finish the tasks in the lessons much quicker than others you might want to check out INLOOP (INteractive Learning-center for Object-Oriented Programming) Inloop allows you to practice Java by solving tasks that will be tested and evaluated automatically.

You will have to be connected to the TUD WiFi (eduroam) or use a VPN to access INLOOP.
I recommend you start with the "Basic" tasks and work your way up to the "Lesson" tasks.

Recomended Tasks in the "Basic" category:

  1. Classes and Objects - Taxi
  2. Java and Functions
  3. String Processing - Leetspeek
INLOOP (only accessable inside the university network) https://inloop.inf.tu-dresden.de/
